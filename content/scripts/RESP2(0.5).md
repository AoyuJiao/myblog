# RESP2(0.5)计算器

这是一个基于气相和溶剂相的电荷文件计算RESP2(0.5)电荷的小脚本。需要Multiwfn生成的`.chg`格式文件作为输入，计算完成后在脚本所在路径输出一个`mol.chg`文件。

两个用于计算的`.chg`文件的原子数应当相等，如不相等将以第一个气相数据为准。

## `.chg`文件的结构示例

从左至右依次为元素、X坐标、Y坐标、Z坐标、电荷。

```
C    41.813972    27.155541    -14.372930    0.3820969318
C    40.820259    27.097566    -13.210756    -0.2451026789
C    39.135587    27.568137    -15.000833    -0.7144861268
C    39.488245    27.698465    -13.604434    0.7242020889
```

## 使用方法

首先，给脚本赋予执行权限：

```bash
chmod +x ./RESP2_0.5.sh
```

然后，运行脚本：

```bash
./RESP2_0.5.sh
```

按照提示将`.chg`文件拖入命令行窗口，并回车确认即可。
